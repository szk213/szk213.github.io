# Vi
## Command

コピー ＆ ペースト

## コマンドモード
v: 選択
V: 行選択
p: ペースト
y: コピー(ヤンク)　※未選択の場合、カーソルの行をコピー
u: アンドゥ
Ctr + r: リドゥ
